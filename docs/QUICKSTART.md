# 5G-MPQUIC-FEC-Fusion 使用指南

## 快速开始

### 编译项目

```bash
# 创建构建目录
mkdir build && cd build

# 配置项目
cmake ..

# 编译
make -j$(nproc)
```

### 运行演示程序

```bash
# 从构建目录运行
./bin/demo_example
```

## 演示程序说明

演示程序展示了以下功能：

### 1. FEC编码演示
- 使用Reed-Solomon纠删码编码数据
- 配置：4个数据块 + 2个冗余块
- 演示：模拟丢失2个数据块后的恢复过程

### 2. 路径调度演示
- 基于在线凸优化(OCO)算法的多路径调度
- 模拟3条不同质量的5G链路
- 动态调整路径权重以优化传输性能

### 3. 缓冲区管理演示
- 零拷贝缓冲区实现
- 内存池管理
- 移动语义支持

### 4. 综合场景演示
- 结合FEC和路径调度的完整传输场景
- 模拟网络丢包和FEC恢复
- 展示系统的容错能力
